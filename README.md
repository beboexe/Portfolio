# Muneeb's Portfolio

This repository contains my personal portfolio website, designed to showcase my web development journey and the projects I have built using HTML, CSS, and JavaScript. The portfolio highlights both clone applications and utility tools, reflecting my learning process and practical application of modern web technologies.

The site features a clean, responsive layout with a navigation bar, hero section, project showcase, and contact area. Each project is presented with its name, a short description, and a consistent design style to ensure clarity and professionalism. The portfolio is optimized for desktop, tablet, and mobile devices, making it accessible across different screen sizes.

## Projects Included
- Netflix Clone – streaming UI with responsive design
- Spotify Clone – music player interface
- Digital Clock – real-time clock built in JavaScript
- BMI Calculator – health utility app
- Joke Maker – random joke generator
- Text Utils – text manipulation tools
- Weather App – live weather data by city

## Features
- Responsive design using Flexbox and Grid
- Modern UI with consistent styling
- Organized project showcase section
- Contact section with email and social links
- Built entirely with HTML, CSS, and JavaScript

This portfolio represents my growth as a developer and serves as a central hub for my projects. It is intended to demonstrate my skills in front-end development, problem solving, and UI design. Future updates will include additional projects and refinements as I continue to learn and expand my expertise.
